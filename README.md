## Стек
* HTML
* CSS
* React.js
* Node.js
* Express
* MongoDB

## Запуск проекта
Для запуска проекта:
1. Клонировать репозиторий
2. Установить зависимости `npm install`
3. Запустить проект `npm start`
4. Проект запускается на http://localhost:3000/
